# Hospital Patient Analysis 🏥

## Overview

A data analysis project examining patient admissions, costs, and readmission rates across hospital departments, using Python, SQL, and data visualization. Built using real-world healthcare data structures from my background as a certified healthcare assistant (UK).

## Features

- Patient data analysis across 3 departments (Cardiology, Diabetes, Psychiatry)
- Cost calculation per patient and department
- Readmission rate tracking
- SQL queries for hospital database analysis
- Data visualization with charts
- Export to Excel reports

## Technologies Used

- **Python 3**
- **Pandas** — data manipulation and analysis
- **Matplotlib** — data visualization
- **SQL** — hospital database querying
- **OpenPyXL** — Excel export

## Key Insights

- Cardiology had the highest total cost (£8,500)
- Average length of stay: Diabetes (6 days), Cardiology (5 days), Psychiatry (2 days)
- 40% readmission rate identified across departments

## Note on Data

This project uses synthetic data generated with realistic clinical value ranges. No real patient data is used, in line with healthcare data privacy principles (GDPR / NHS information governance).
